终版转移到前一个文件夹.py              这个代码可以将文件夹中的前96个文件转移到前一个文件夹中用于gnss接收机中原始数据重分组因为原文件夹是以北京时为基准，而文件夹中包含了前一天的原始文件，而使用gfzrnx时他只能识别世界时的日期，一个文件夹中有288个文件，每五分钟一个文件，前96个文件正好是8个小时对应的就是UTC+8

将.bdat转o文件后合并并改名为一天.py          这个代码是将上面的那个文件夹中的.bdat文件先转换成o文件，然后再用gfzrnx工具将单日所有的o文件合并成一个单日文件并改名字成GNSSREFL工具能够识别的o文件，文件夹的结构是例如：9753文件夹下有20250502，20250503...,在20250502文件夹中202505030800.bdat，202505030801.bdat，202505030802.bdat。这个代码能够自动识别20250502文件夹的名称，并提取年月日，最为最后单日文件的名称例如975300CHN_R_20251220000_01D_05S_MO.rnx  这个代码需要rtklib以及gfzrnx两个程序，rtklib是一个文件夹，gfzrnx是单个应用程序。在代码最后CONVBIN_EXE中选择rtklib文件夹中的convbin_exe的路径，gfzrnx就选择那个单应用的文件路径。除此之外还有原始数据根目录和你的测站名都在下面记得修改。
